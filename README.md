# Kursova_NVBD_Variant_01

# Курсова робота з НВБД

## Тема
Аналітична система деканату з використанням OLAP та BI-технологій

## Технології
- MS SQL Server
- SSIS
- SSAS
- SSRS
- GitHub

## Структура репозиторію
- Database — SQL-скрипти створення БД
- DataGeneration — генерація тестових даних
- SSIS_Projects — ETL процеси
- SSAS_Projects — OLAP куб
- SSRS_Projects — звіти та дашборд
- Documentation — курсова робота (PDF)

## Реалізовані звіти
- Табличний звіт студентів
- Матричний звіт оцінок
- Діаграми (Column, Pie, Line, Bar)
- Інтерактивний Dashboard
